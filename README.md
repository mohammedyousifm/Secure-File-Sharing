# Secure File Sharing Project

## Overview
Secure File Sharing is a web-based platform designed to provide users with a secure and efficient way to share files, communicate, and collaborate. The platform prioritizes user privacy and data security by employing encryption mechanisms for files and messages. It also offers user authentication and profile management, ensuring a smooth and secure experience.

---

## Technologies Used
- **Backend:** PHP (Laravel Framework)
- **Frontend:** Tailwind CSS, Bootstrap (to be replaced soon), and JavaScript (with plans to integrate additional JS libraries).
- **Database:** MySQL
- **Encryption:** OpenSSL for secure communication and file encryption.

---

## Current Features
1. **User Authentication:**
   - Users can create accounts and log in securely.
   
2. **Friendship System:**
   - Users can send and accept friend requests to connect with others.

3. **File Sharing:**
   - Users can send and receive files.
   - All files are encrypted on the server for enhanced security.

4. **Chat System:**
   - Users can chat with other authenticated users.
   - Messaging is encrypted using OpenSSL for privacy.

5. **Secure Communication:**
   - Users can send messages and share files only with connections that have accepted friendship requests.

6. **User Profile:**
   - Each user has a profile that includes:
     - Profile picture.
     - Name and other details such as position.
     - Statistics, including the number of files sent, received, or deleted.

---

## Future Goals
1. **File Scanning:**
   - Develop a function to scan files for viruses or malware before they are uploaded or stored on the server.

2. **File Preview:**
   - Allow users to preview files (e.g., PDFs, images, or text documents) directly within the platform without needing to download them.

3. **Encryption Enhancements:**
   - Strengthen encryption algorithms and explore additional security features to safeguard user data.

4. **Improved User Experience:**
   - Replace Bootstrap with a more modern and efficient CSS framework.
   - Introduce advanced JavaScript libraries to enhance interactivity.

5. **Activity Logs:**
   - Add detailed activity logs for users to track their actions, such as file uploads, deletions, and chat history.

6. **Access Controls:**
   - Implement role-based access control (RBAC) to provide different permissions for admins and regular users.

7. **Scalability:**
   - Optimize the platform for scalability to handle a growing user base and larger file uploads efficiently.
